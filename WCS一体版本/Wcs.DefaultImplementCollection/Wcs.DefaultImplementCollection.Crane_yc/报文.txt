
 全部报文
 ------------------------------------------------------
 HA  查询堆垛机状态						<HA>
 LA  应答 堆垛机状态						<LA	00	111	222	3	4	5	6666	7	88888888	char3
 HB  发任务
 LB  应答 接收任务状态的确认
 HE  紧急停机
 HC  故障复位 (约定任务号 GOT00000)
 HP  回原点   (约定任务号 GOT00001)

 报文 开始、结束字符
 ------------------------------------------------------
 GOT <-> WCS 报文 开始字符 '<';       结束字符 '>';
 GOT <-> CRN 报文 开始字符 '(char)2'; 结束字符 '(char)3';
 
 所有 通讯方向的报文
 ------------------------------------------------------
 GOT -> CRN  <HA>
 GOT <- CRN  <LA + 货叉左右位(1) + 货叉高低位(1) + 站点位置(1) + 错误码(4) + 事件(1) + 任务号(8)>
 GOT -> CRN  <HB 起点排列层(8) + 终点排列层(8) + 动作命令(2) + 任务号(8)>
 GOT <- CRN  <LB 接收任务状态(1) + 任务号(8)>

 GOT -> CRN  <HE>
 GOT -> CRN  <HC>
 GOT -> CRN  <RE>
 GOT -> CRN  <HP>

 GOT <- WCS  <HA>
 GOT -> WCS  <LA 堆垛机号(4) + 地操状态(1) + 垛机状态(2) + 错误码(4) + 事件(1) + 任务号(8) + 垛机列层(6)>
 GOT <- WCS  <HB 任务号(8) + 起点排列层(8) + 终点排列层(8) + 条件(1) + 配置(1)>
 GOT -> WCS  <LB 接收任务状态(1) + 任务号(8)>

 地操状态
 ------------------------------------------------------
 0、非联机模式 (半自动、手控)
 1、联机模式 (自动)
 2、暂停 (故障复位后状态)
 3、堆垛机故障

 01、回原点
 02、无货待命
 03、有货待命
 04、无货运行
 05、有货运行
 06、取货
 07、放货
 08、故障停机
 09、故障复位
 10、???
 11、未连接 ???
 12、手操盒模式

 ------------------------------------------------------
 1、中位
 2、左
 3、左极限
 4、右

 ------------------------------------------------------

 1、高位

 ------------------------------------------------------
